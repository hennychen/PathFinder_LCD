本模块由两个互补子项目组成：
- `main/`（ESP-IDF C 应用）：以 `main.c` 为入口，按职责拆分为 `drivers/`（AHT20/BMP280/MPU6050/UV ADC 硬件驱动）、`sensor_manager.c`（传感器聚合）、`motion_engine.c`（姿态解算）、`emote_engine.c`（表情动画调度）、`ble_gatt_server.c`（BLE 服务暴露）、`web_portal.c`（配置 Web 页面）、`provision_screen.c`（配网屏）等文件，并通过 `app_emote_assets.c` 加载预编译的表情资源。
- `managed_components/`：通过 IDF Component Manager 引入 LVGL、esp_lv_eaf_player、esp_mmap_assets、esp_new_jpeg、cJSON 等组件，供嵌入式端 UI 与多媒体播放使用。
- 跨端契约：BLE GATT 是两端唯一运行时通信通道，嵌入式侧在 `ble_gatt_server.c` 中定义特征值并推送传感器/表情帧，Flutter 侧消费这些特征更新驱动 UI；构建期通过 `partitions.csv` 划分 OTA/ NVS/ assets 分区，`sdkconfig.defaults` 统一 Flash 大小与外设开关。
- 重构意图：作者计划将嵌入式端进一步拆出 `app/`（UI 导航、LED 规则引擎、传感器 Hub）与 `ui/`（各 lvgl 页面），当前代码仍集中在 `main/` 扁平结构中。
