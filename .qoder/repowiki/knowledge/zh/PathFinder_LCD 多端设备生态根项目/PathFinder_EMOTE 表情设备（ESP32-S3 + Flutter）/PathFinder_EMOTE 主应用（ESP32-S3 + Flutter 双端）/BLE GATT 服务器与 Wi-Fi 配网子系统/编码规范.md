- 每个 .c 文件顶部用 `static const char *TAG = "xxx";` 定义日志标签，所有日志通过 `ESP_LOGI/W/E` 输出。
- 对外 API 采用 `*_init()` 单例初始化 + 返回 `esp_err_t` 的错误码约定，内部以 `s_initialized` 守卫重复初始化。
- BLE 二进制帧以小端辅助函数 `put_u16le/put_u32le` 写入固定偏移，并通过注释在文件头声明各特征值的字节布局。
- GATT 特征值访问统一走 `gatt_chr_access(conn_handle, attr_handle, ctxt, arg)` 单入口 switch 分发，按 `attr_handle` 区分不同特征值。
- 可插拔回调通过 `static xxx_cb_t s_xxx_cb = NULL;` + 对应 `register_xxx_cb(cb)` 暴露，避免直接 include 上层模块。
