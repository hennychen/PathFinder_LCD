- 所有公共 API 使用 `extern "C"` 包裹并遵循 `#ifdef __cplusplus` 保护，保证 C/C++ 互操作。
- 控件结构体均以 `lv_obj_t obj;` 作为首成员，并通过 `lv_obj_class_t` 描述类行为（constructor/destructor/event_cb/instance_size），形成 C 语言下的面向对象继承模式。
- 可插拔子系统（显示驱动、输入设备、渲染后端、图像解码器）统一采用“驱动结构体 + 函数指针回调”的注册式接口，如 `lv_disp_drv_t.flush_cb`、`lv_draw_ctx_t.draw_rect`、`lv_img_decoder_t`.decode_cb。
- 功能模块通过 `LV_USE_XXX` 宏进行条件编译，头文件顶部用 `#if LV_USE_BTN != 0` 等守卫隔离可选代码。
- 枚举常量统一以 `LV_STATE_`、`LV_PART_`、`LV_OBJ_FLAG_`、`LV_STYLE_PROP_` 等前缀命名，并按 16 位/32 位位域组合使用。
