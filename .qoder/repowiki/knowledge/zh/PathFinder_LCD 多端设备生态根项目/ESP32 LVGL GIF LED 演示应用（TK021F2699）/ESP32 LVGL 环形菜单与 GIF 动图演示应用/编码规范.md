- 对外部可编译资源（图片、字体、GIF 帧）统一以 `ui_img_*` / `ui_font_*` / `gif/*` 前缀命名，并在 `CMakeLists.txt` 的 SRCS 中逐条列出而非通配。
- 跨模块共享的全局句柄（如 `ws2812_strip_handle_t ws2812_handle`、`SemaphoreHandle_t lvgl_mux`）在 main 中定义并通过 `extern` 在其他源文件中引用，避免头文件污染。
- 所有 LVGL API 调用在进入/退出时成对包裹 `example_lvgl_lock(-1)` / `example_lvgl_unlock()`，确保 LVGL 线程安全。
- 硬件相关常量（GPIO 编号、像素时钟、分辨率、背光极性）集中在 main 顶部以 `EXAMPLE_*/CONFIG_EXAMPLE_*` 宏集中管理，便于裁剪不同硬件变体。
