- 所有 UI 组件统一采用 `ConsumerWidget` / `ConsumerStatefulWidget` 模式，通过 `ref.watch(motionStreamProvider)` 订阅 IMU 流，而非手动管理 StreamSubscription。
- 数值展示复用 `_MotionDataCard` 胶囊卡片样式（圆角 16、半透明背景 + 边框），单位文本右对齐并降低字重，保持绿色 EMA 风格一致。
- 自定义绘图 widget 使用独立的 `CustomPainter` 子类并在 `shouldRepaint` 中精确比较输入字段（如 pitch/roll），避免全量重绘。
- 高频刷新图表通过本地 `List<ImuSnapshot>` 缓冲 + `maxPoints` 截断，配合 `RepaintBoundary` 隔离，防止整屏重建。
