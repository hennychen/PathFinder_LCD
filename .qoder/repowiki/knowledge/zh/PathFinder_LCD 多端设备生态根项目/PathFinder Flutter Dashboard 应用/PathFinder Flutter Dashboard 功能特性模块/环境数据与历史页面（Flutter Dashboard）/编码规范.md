- Feature 入口文件采用 `<feature>_screen.dart` 命名并以 `ConsumerWidget` 暴露，通过 `ref.watch` 订阅 Riverpod Provider 驱动 UI 更新。
- 数值指标卡片复用 `shared/widgets/metric_card.dart`，按维度传入 value/unit/label/color/backgroundColor 参数，避免在每个 Screen 中重复布局。
- 阈值→配色映射集中在 Screen 内私有方法（如 `_tempColor`、`_humidityColor`），将业务规则与 UI 构建解耦，便于后续迁移到 Provider。
- 图表组件独立拆入 `widgets/` 子目录并自管本地缓冲（如 `EnvLineChart` 的 `_buffer` + maxPoints 滑动窗口），对外仅暴露无参构造函数。
