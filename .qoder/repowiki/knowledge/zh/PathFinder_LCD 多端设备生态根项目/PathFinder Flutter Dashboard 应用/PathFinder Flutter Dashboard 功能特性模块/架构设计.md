本目录按业务域切分为 connection/emote/environment/motion/history/wifi 六个子模块，每个子模块提供独立的 `*_screen.dart` 作为路由入口，并在各自 `widgets/` 下封装专用图表与控件（如 `imu_wave_chart.dart`、`attitude_indicator.dart`、`env_line_chart.dart`）。跨模块的耦合点集中在两个层面：
- 共享 UI 壳层：重构后的 `lib/app/app.dart` 使用 `IndexedStack` 承载环境/运动/表情三个 Tab，并将 BLE 连接入口从独立 Tab 迁移至 AppBar 右上角的 `BleStatusChip`，使所有业务页共享同一连接状态。
- 共享状态：各 Screen 通过 `ConsumerWidget` / `Provider` 订阅 BLE 连接状态与传感器数据流，而非直接互相引用，从而保持子模块间松耦合。
