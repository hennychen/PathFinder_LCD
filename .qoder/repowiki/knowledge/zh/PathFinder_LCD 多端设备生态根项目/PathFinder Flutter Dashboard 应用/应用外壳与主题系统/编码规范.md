- 主题常量集中在 `AppColors` 抽象类中以 `static const` 暴露，避免魔法数字散落各处。
- 字体样式集中在 `AppTypography` 抽象类中以 `static const TextStyle` 暴露，按 display/headline/title/body/label/metric 语义命名。
- 主题组装采用工厂函数 `appTheme()` 返回 `ThemeData`，而非直接写死在 MaterialApp 构造参数中。
- 全局 UI 壳使用 `ConsumerStatefulWidget` 包装，以便在 `build` 中通过 `ref.watch` 订阅 BLE 状态并驱动 AppBar actions 的可见性。
