- `lib/main.dart` 仅做最薄封装：`runApp(ProviderScope(child: PathfinderApp()))`，将 Riverpod 的 ProviderScope 作为根 Widget。
- `lib/app/app.dart` 中的 `PathfinderApp` 是全局唯一 `ConsumerStatefulWidget`，负责：
  - 通过 `MaterialApp(theme: appTheme())` 注入主题；
  - 用 `IndexedStack` 持有环境/运动/表情三个去 Scaffold 化的页面，配合 `NavigationBar` 实现 3-Tab 切换；
  - 在共享 `AppBar.actions` 中放置 `BleStatusChip`（BLE 连接状态芯片）、WiFi 设置按钮与条件显示的历史数据按钮，并通过 `Navigator.push` 路由到 `HistoryScreen` / `WifiSetupScreen`。
- `lib/app/theme/` 为纯主题资源层，无业务逻辑：
  - `app_colors.dart` 以 `abstract class AppColors` 暴露暗色背景、四象限强调色（环境青/运动绿/警告琥珀/紧急红）及 EMA 胶囊半透明背景；
  - `app_theme.dart` 的 `appTheme()` 工厂函数组合 `ColorScheme.dark`、`CardThemeData`、`NavigationBarThemeData`、`AppBarTheme`、`DividerThemeData`，统一引用 `AppColors`；
  - `app_typography.dart` 提供 display/headline/title/body/label/metricValue/metricUnit 等静态 `TextStyle`。
- 依赖方向单向：`main → app → theme`，页面与共享组件通过 `../features/*` 和 `../shared/widgets/*` 反向被 app 引入，theme 不依赖任何 feature。
